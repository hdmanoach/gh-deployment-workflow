# GH Deployment Workflow

Ce projet démontre le déploiement automatique d’un site web statique sur GitHub Pages en utilisant GitHub Actions.

## Fonctionnement

- Chaque modification du fichier `index.html` sur la branche `main`
- Déclenche automatiquement un workflow GitHub Actions
- Qui déploie le site sur GitHub Pages
